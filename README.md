# Fidelity Clone

A comprehensive portfolio management interface mimicking Fidelity's features with transaction history and analysis.

## What It Is

Fidelity Clone is a full-featured investment dashboard that replicates Fidelity's interface and functionality. Import your positions and transaction history from CSV, view detailed portfolio summaries including cost basis and gains, and explore individual stock performance. The app includes account management across multiple account types, historical balance charts, activity logs, and investment theses for individual holdings, all with live price updates.

## Tech Stack

- React 19 with TypeScript
- Vite for fast development
- Recharts for financial visualizations
- PapaParse for CSV parsing
- Yahoo Finance for price data
- TSX for TypeScript scripts

## Getting Started

```
npm install
npm run dev
```

Use the Import tab to load your portfolio data from CSV files, then explore your positions, transactions, and performance across the various dashboard views.
